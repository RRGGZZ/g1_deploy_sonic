# 部署命令速记

## 🎯 一句话部署（复制即用）

```bash
# 最简单 - 一键脚本
cd ~/Downloads/GR00T-WholeBodyControl/gear_sonic_deploy/reference/example_scalelab/qwen_audio_new && ./deploy_qwen_audio_new.sh real

# 或者标准方式
cd ~/Downloads/GR00T-WholeBodyControl/gear_sonic_deploy && ./deploy.sh real --motion-data reference/example_scalelab/qwen_audio_new --motion-audio-dir reference/example_scalelab/qwen_audio_new/audio_aligned_16k_en
```

## 📋 完整命令参考

| 用途 | 命令 |
|------|------|
| **真机部署** (推荐) | `./deploy.sh real --motion-data reference/example_scalelab/qwen_audio_new --motion-audio-dir reference/example_scalelab/qwen_audio_new/audio_aligned_16k_en` |
| **仿真测试** | `./deploy.sh sim --motion-data reference/example_scalelab/qwen_audio_new --motion-audio-dir reference/example_scalelab/qwen_audio_new/audio_aligned_16k_en` |
| **快速脚本** | `cd reference/example_scalelab/qwen_audio_new && ./deploy_qwen_audio_new.sh real` |
| **指定接口** | `./deploy.sh eth0 --motion-data reference/example_scalelab/qwen_audio_new --motion-audio-dir reference/example_scalelab/qwen_audio_new/audio_aligned_16k_en` |

## 🎮 运行时快捷键

```
T → 播放   N → 下一个   P → 暂停   R → 重新启动   Q → 退出
```
